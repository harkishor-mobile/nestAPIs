## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

# Command for initialize project

nest new my-nest-app --package-manager pnpm

# Install Dependencies

pnpm install

## Compile and run the project

# development

pnpm run start

# watch mode

pnpm run start:dev

# production mode

pnpm run start:prod

## Run tests

# unit tests

pnpm run test

# e2e tests

pnpm run test:e2e

# test coverage

pnpm run test:cov
